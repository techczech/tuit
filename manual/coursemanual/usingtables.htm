To create a table like this, use the

<table border="1" width="100%">

Marks the beginning of a table and defines its characteristics.

  <tr>

Marks the beginning of a row.

    <td width="33%">Jsem z</td>

Marks the beginning and end of a column in row and thus defines the content of a cell. Not only width but height can also be defined. Other parameters such as color can also be set up.

    <td width="33%">Prahy</td>

    <td width="34%">Praha</td>

  </tr>

Marks the end of a row.

  <tr>

 

    <td width="33%">Petr je z</td>

Width could be specified in pixels as well.

    <td width="33%">Brno</td>

 

    <td width="34%">Brna</td>

(Notice: The HTML tags can be both caps and small letters.

  </tr>

i.e. <p> and <P> do the same thing.)

</table>

Marks the end of a table
